## Install :

Clone Repo Git : https://github.com/Imo999/dapp_v0.0

```zsh
% git clone git@github.com:Imo999/dapp_v0.0.git
% cd dapp_v0.0
```

## Install dependencies :

```zsh
% yarn install
```

## Run DEV Mode :

```zsh
% yarn start
```
